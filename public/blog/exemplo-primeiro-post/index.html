Post de exemplo: Checklist rápido de comunicação cultural

Modelo para novos artigos: data, tags, imagem de capa opcional e conteúdo em Markdown.

Este é um post de exemplo criado pelo CMS. Você pode editar este texto pelo painel /admin, atualizar a imagem de capa e salvar para gerar a página HTML automaticamente no deploy.

O que ajustar

  • Troque o título e o resumo acima pelo tema do seu artigo.
  • Substitua a data no formato YYYY-MM-DD.
  • Adicione ou remova tags conforme o assunto.
  • Caso não tenha imagem, deixe o campo de capa vazio.

Estrutura em Markdown

Use títulos, listas, links e imagens normalmente:

  • Defina o objetivo do post na introdução.
  • Liste passos ou checklists para o leitor aplicar.
  • Inclua links úteis e referências de imprensa ou redes.
